# 🌿 Potato Disease Detection Using CNN

An AI-powered Deep Learning project that detects potato leaf diseases using Convolutional Neural Networks (CNN) and Computer Vision techniques.

##  Project Overview

This project classifies potato leaf images into three categories:

- Potato Early Blight
- Potato Late Blight
- Healthy Potato Leaf

The model is trained using the PlantVillage dataset and deployed using Streamlit for real-time predictions.

##  Features

✅ Detects potato leaf diseases from uploaded images  
✅ Deep Learning model using CNN  
✅ Streamlit web application  
✅ Real-time prediction system  
✅ Confidence score display  
✅ Disease cause and prevention information  
✅ Clean and attractive UI design  


##  Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Streamlit
- Jupyter Lab

---

##  Dataset

Dataset used:
PlantVillage Dataset from Kaggle

Only sample images are uploaded in this repository to keep the project lightweight.


##  Model Architecture

The project uses a Convolutional Neural Network (CNN) for image classification.

### Steps:
1. Image Preprocessing
2. Image Resizing
3. Normalization
4. CNN Model Training
5. Model Evaluation
6. Disease Prediction


##  Model Accuracy

✅ Training Accuracy: 97%  
✅ Testing Accuracy: 96.23%


##  Streamlit Web App

The web app allows users to:

- Upload potato leaf images
- Predict disease instantly
- View confidence score
- Read disease prevention information

##  How to Run Project

### 1. Install Required Libraries

pip install tensorflow streamlit opencv-python numpy matplotlib pillow

### 2. Run Streamlit App

streamlit run app.py
